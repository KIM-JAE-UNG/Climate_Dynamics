## 🌊 소개

**DEEP DROP**은 클래식 테트리스를 심해 낚시 테마로 재해석한 웹 기반 게임입니다.  
블록이 물고기가 되고, 라인 클리어가 곧 낚시가 됩니다.  
레벨이 올라갈수록 깊은 바다로 내려가며, 도감을 완성해보세요!

## 🐟 물고기 도감

| 블록 | 물고기 | 무게 |
|---|---|---|
| I (━━━━) | 🐍 장어 | 3.2kg |
| O (■■) | 🐡 복어 | 1.5kg |
| T (┳) | 🦈 가오리 | 4.8kg |
| S (⌐¬) | 🐴 해마 | 0.8kg |
| Z (¬⌐) | 🦐 새우 | 0.5kg |
| J (┏) | 🐟 고등어 | 2.1kg |
| L (┓) | 🐙 문어 | 3.5kg |

## 🌊 수심 시스템

10줄 클리어마다 더 깊은 바다로 이동합니다.

1. 🏖️ 얕은 바다
2. 🌊 연안
3. ⛵ 외해
4. 🌑 심해 입구
5. 🐙 심해
6. 🌋 해저 화산
7. 👀 마리아나 해구

깊어질수록 배경이 어두워지고 블록 속도가 빨라집니다!

## ✨ 주요 기능

- **물고기 블록** — 각 테트리스 블록이 다른 해양 생물
- **도감 시스템** — 잡은 물고기를 수집하고 마릿수 확인
- **수심 시스템** — 레벨업 시 점점 깊은 바다로, 배경이 실시간 변화
- **캐치 알림** — 라인 클리어 시 잡은 물고기와 무게 표시
- **수중 배경** — 물방울, 광선 애니메이션으로 해저 분위기
- **게임 오버 통계** — 귀항 시 잡은 물고기 종류별 통계
- **홀드 / 미리보기 / 고스트 피스** — 표준 테트리스 기능
- **반응형 + 모바일 지원** — 터치 컨트롤 포함

## 🎮 조작법

| 키 | 동작 |
|---|---|
| `←` `→` | 이동 |
| `↑` | 회전 |
| `↓` | 소프트 드롭 |
| `Space` | 하드 드롭 (투하) |
| `C` | 보관함 (홀드) |
| `P` | 일시정지 |

## 🚀 실행 방법

### 로컬
```
index.html 파일을 브라우저에서 열기
```

### GitHub Pages
1. 이 저장소를 GitHub에 Push
2. Settings → Pages → Source를 `main` 브랜치로 설정
3. `https://<username>.github.io/<repo-name>` 에서 플레이!

## 🛠️ 기술 스택

- **HTML5 Canvas** — 게임 + 배경 렌더링 (듀얼 캔버스)
- **CSS3** — 글로우 이펙트, 애니메이션, 반응형
- **Vanilla JavaScript** — 게임 로직, 파티클, 물리
- 외부 라이브러리 없음 (Zero Dependencies)
